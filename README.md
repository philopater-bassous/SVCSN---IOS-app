# 📱 SVCSN – Saint Vincent College Sports Network

SVCSN is a native iOS application built with **SwiftUI**, **Firebase**, and **Xcode** that provides the Saint Vincent College community with real-time athletics information and campus updates. The app serves as a central hub for live scores, team rosters, schedules, announcements, and other school-related content.

---

## ✨ Features

- 🏆 Live game scores and results
- 📅 Team schedules
- 👥 Player rosters
- 📢 School announcements
- 🔥 Real-time updates powered by Firebase
- 📱 Modern SwiftUI interface
- ⚡ Fast and responsive user experience

## 🚀 Getting Started

### Prerequisites

- macOS
- Xcode 16+
- Swift 6+
- Firebase Project

### Installation

1. Clone the repository

```bash
git clone https://github.com/philopater-bassous/SVCSN---IOS-app.git
```

2. Open the project

```text
SVCSN.xcodeproj
```

3. Configure Firebase

- Create a Firebase project
- Download your `GoogleService-Info.plist`
- Add it to the Xcode project

4. Build and Run

## 🎯 Purpose

SVCSN was developed to improve how students, families, alumni, coaches, and fans stay connected with Saint Vincent College athletics. Instead of searching multiple websites or social media pages, users can access schedules, rosters, live scores, and announcements from one mobile application.
